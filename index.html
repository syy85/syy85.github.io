快速搭建个人博客

博客搭建

程序安装

安装 Hexo 前，需安装以下程序：

  • Node.js (建议使用 Node.js 12.0 及以上版本)
  • Git（版本不要过于老旧即可）
  • Node.js安装教程
  • Windows系统Git安装教程

注意：由于 Git 服务器在国外 ，在 Git 官网下载的话速度极其缓慢。可自行百度使用国内资源下载.

网站搭建

以上程序成功安装后，使用 nmp 安装 Hexo

$ npm install -g hexo-cli

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

$ hexo init <folder>
$ cd <folder>
$ npm install

新建完成后，指定文件夹的目录如下：

.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

本地博客查看

在 hexoblog 文件夹右击鼠标打开 Git Bash Here 输入 hexo s命令即可本地博客网址

$ hexo s
INFO  Validating config
INFO  Start processing
INFO  Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

如上图所示：localhost:4000 即为本地博客网址。
